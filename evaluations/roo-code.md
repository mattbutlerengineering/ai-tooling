## What it does

An open-source autonomous coding agent for VS Code — a fork of Cline offering customizable agent modes. The repo was archived on 2026-05-15 when the team sunset the OSS extension in favour of a paid cloud product (roocode.com now redirects to roomote.dev).

## How we tested it

**Evidence:** SOURCE-ONLY

We did **not** install or run this tool. This evaluation is source-grounded only: it rests on
GitHub repo metadata (archived flag, last push, licence) and on published statements from the
project itself about why it was archived and where its users should go. Archived status and the successor lineage were established from repo metadata and the Roo→Kilo migration guide; the extension was never installed or exercised.

That is sufficient for the verdict below, because the verdict turns on *maintenance status and
succession*, not on the tool's behaviour — a question metadata answers directly. It would not be
sufficient to support an ADOPT, and this eval does not offer one.

## Successor

`kilocode` (Kilo-Org/kilocode) and `cline` (cline/cline) — both already in CATALOG.md

## Verdict

**SKIP** — archived, and its successor is already catalogued. Roo Code's OSS extension was wound down in favour of a paid cloud product; the team points users to Kilo Code (a direct git-history fork) and to Cline. Both `kilocode` and `cline` are already CATALOG entries, so repointing this row would duplicate them. Nothing here to adopt that the catalog does not already carry.
