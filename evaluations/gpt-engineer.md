## What it does

A CLI that generates a codebase from a natural-language prompt — one of the earliest agentic codegen tools. Archived on 2025-05-14. Its own repo description reads: 'CLI platform to experiment with codegen. Precursor to: https://lovable.dev'.

## How we tested it

**Evidence:** SOURCE-ONLY

GitHub repo metadata (archived flag, last push, licence) and on published statements from the

none in OSS — the effort became the closed-source commercial product Lovable

## Verdict

**SKIP** — archived, and its successor is closed-source. The project's effort moved into Lovable, a commercial for-profit product, which is not an OSS successor and cannot be adopted under this catalog's licence bar. The repo's own README redirects hackable-CLI users to Aider, which is a recommendation rather than a lineage successor.
