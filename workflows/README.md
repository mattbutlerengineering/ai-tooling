# AI Dev Workflow Research

Dated snapshots of what developers are actually doing with AI coding agents — common patterns, emerging practices, and where the tooling is heading. Each file covers 3-5 workflows with concrete pros, cons, and suggestions tied back to this repo's catalog.

## Format

Each file opens with a one-line theme and a `Sources:` line of cited links, then covers 3-5 workflows. Each workflow has a **bold title** and four labeled parts:

- **What it is** — concrete description of the pattern
- **Pros** — real benefits with specifics
- **Cons** — real drawbacks, not disclaimers
- **Suggestions** — tool or process recommendations, tied to CATALOG.md / WORKFLOW.md where relevant
