## What it does

## How we tested it

**Evidence:** SOURCE-ONLY

We did **not** install or run this tool. This evaluation is source-grounded only: it rests on
GitHub repo metadata (archived flag, last push, licence) and on published statements from the

succession*, not on the tool's behaviour — a question metadata answers directly. It would not be

## Verdict

**SKIP** — archived with no successor. Development is paused rather than migrated: the team shipped no successor repo and made no continuation announcement. Community forks are collected at `voideditor/void-forks`, but none is official or endorsed. An unmaintained editor is not something to build a workflow on.
